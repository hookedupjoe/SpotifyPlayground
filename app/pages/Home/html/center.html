Spotify API and Web Player
Spotify Developer Playground
To get started, create a new spotify application in the developers console and enter the client id (no need to provide your secret key).
This is a developer playground to work with the API and to provide a complete example of using the Spotify API with a serverless hosted website. This site is hosted on GitHub and is 100% client side.
How it works: This uses the clientid only, no secret key is needed for this implementation.
There is no server and no way for a person to get the clientId outside of your own computer. The clientId can be removed anytime, this is providing an application access to your account in a way that only you - on this browser can control.
This article explains it all and has links to everything implemented here.
Once you have a client id, use the button below. That saves it to localStorage so when you come back, it remembers you. You can clear storage or set them to blank if you like, but this should only be used on your own private computer.

About The Web Player
Official documentation on the player
The Web Player allows you to play Spotify directly on this site. Just like any device, it can be controlled by any Spotify client while connected. Pretty cool!
How to use it here ...
Connect the player by clicking Connect Player button.
Once connected, turn on the player by clicking the newly available Toggle Player button.
Select this web player in Spotify on any device.
Select some music such as a playlist.
Hit play on the remote player, you can then control the player using the buttons below.
Load up a playlist and hit play.
